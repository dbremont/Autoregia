# Inclusion-Exclusion Principle

Proof:

For $n$ sets $A_1, A_2, \dots, A_n$:

$\left| \bigcup_{i=1}^n A_i \right| = \sum_{k=1}^n (-1)^{k+1} \sum_{1 \leq i_1 < \dots < i_k \leq n} |A_{i_1} \cap \dots \cap A_{i_k}|$

- Sum over all **non-empty subsets** of indices.
- **Alternate signs**: include (odd $k$), exclude (even $k$).
