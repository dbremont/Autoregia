Personal Viable System Model

Autoregia

A self-management system — a recursive regulatory architecture through which an individual monitors, organizes, and steers perception, ideation, goal-formation, action, and reflection.

Self-management is the praxis of constructing and sustaining an agential hierarchy, so that operative processes evolve from implicit, unexamined routines into explicit, intelligible, and systematically steerable structures.
Autoregia decomposes self-management into a set of cooperating systems, each mapped to a level of the Viable System Model. Some are materialized as dedicated tools developed in this workspace; others are satisfied by existing tools or documents.
