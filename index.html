Agent Self-Representation System

ASRS

Binds the World boundary, the external-world Model (PWMS), the Self Model (PSMS), and the Policy System (PPS) into one coherent structure — and checks it against itself for over-allocation, constraint violation, and value drift.

Enter asrs/ → substrate · Self-Model (capabilities)

Personal Technical Object Catalog System

PTOCS

The agent's catalog of the technical objects it relies on (software, services, hardware, references, capabilities, infrastructure) — externalizes capability awareness: what's available, where it lives, how it fits, and where the gaps are. A Self-Model (PSMS) capability facet.
